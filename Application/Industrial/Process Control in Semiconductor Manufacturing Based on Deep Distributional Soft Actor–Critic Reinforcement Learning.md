
# Process Control in Semiconductor Manufacturing Based on Deep Distributional Soft Actor–Critic Reinforcement Learning

## 1. 논문의 핵심 주장과 주요 기여

본 논문은 **반도체 제조의 복잡한 비선형 공정 특성을 모델-프리 제어 기법으로 효과적으로 관리할 수 있음**을 보여주며, 다음과 같은 주요 기여를 제시합니다.[1]

### 주요 기여

**첫째**, DSAC(Distributional Soft Actor-Critic) 학습을 반도체 CMP(화학 기계 연마) 공정에 처음으로 적용하고, 실제 제조 데이터로 검증하였습니다. **둘째**, 1DCNN-LSTM 기반 다층 신경망 가상 계측(Virtual Metrology, VM) 모델을 결합하여, CMP 공정의 고도의 비선형성과 무작위성을 더욱 정확하게 포착했습니다. **셋째**, 산업 데이터로부터 구성한 훈련 환경에서 백색 잡음, 공정 드리프트, MRR(물질 제거율) 변이를 모두 고려한 포괄적인 성능 평가를 제시했습니다. **넷째**, 동적 목표값 추적 시나리오를 포함한 실질적인 적용 가능성을 입증했습니다.[1]

## 2. 해결하고자 하는 문제, 제안 방법, 모델 구조, 성능 향상

### 2.1 핵심 문제 정의

반도체 제조에서 기존의 **선형 모델 기반 제어 방식의 한계**가 주요 문제입니다. 구체적으로:[1]

- **공정 모델의 부정확성**: 선형 제어 모델(MISO, 다입력-단출력)은 실제 CMP 공정의 고도의 비선형성과 시간 변화 특성을 충분히 반영하지 못합니다.
- **시스템 불확실성**: 공정 드리프트(pad 마모), 설비 드리프트, 환경 잡음 등 시스템 불확실성에 대한 동적 적응 능력 부족
- **멀티존 압력 제어의 복잡성**: 서로 다른 영역의 압력이 상호 영향을 미치는 복잡한 상호작용을 모델링하기 어려움
- **제어 제약 조건**: 실제 공정에서는 회전 속도, 압력 등에 엄격한 제약이 존재하나, 기존 연구는 무제약 환경을 가정

### 2.2 제안하는 방법: DSAC-T 기반 제어 프레임워크

#### 2.2.1 가상 계측 모델 설계

논문에서 제안하는 VM 모델은 두 부분으로 구성됩니다:[1]

**VM1 (드리프트 컴포넌트)**: 1DCNN-LSTM 네트워크로 Zone 5의 MRR을 예측합니다:

$$y_{5,t+1} = CMRR + f_{CNN-LSTM}(\mathbf{x}_{t-14:t-5})$$

여기서 $CMRR$은 정규화된 후의 MRR 상수값, $f_{CNN-LSTM}$은 1DCNN-LSTM 함수입니다.[1]

**VM2 (멀티존 압력 컴포넌트)**: MLP 네트워크로 다른 존의 MRR을 계산합니다:

$$y_{i,t} = CMRR + f_{MLP}(y_{5,t}, p_{i,t}, U_t) + \epsilon_t, \quad i \neq 5$$

$$CMRR = f_{CNN-LSTM}(\mathbf{x}_{t-13:t})$$

여기서 $p_{i,t}$는 존 $i$의 압력, $U_t$는 폴리싱 패드 사용 시간입니다.[1]

#### 2.2.2 MDP 공식화

강화학습 에이전트는 다음과 같은 마르코프 결정 과정으로 정의됩니다:[1]

**상태 (State)**:
$$s_t = [u_{t-k}, p_{i,t-k}, y_{5,t-k}, U_t, y_{i,t+1}, e_{i,t+1}, p_{i,t+1}, y_{R,i}^t]$$

**행동 (Action)**:
$$a_t = \Delta p_i$$
압력 변화를 결정하는 이산 행동 공간입니다.

**보상 함수 (Reward Function)**:
$$r_t = -\sum_{i=1}^{7} \alpha_i |e_i| - \beta(\max e_i - \min e_i) - \gamma |a_i - a_{i-1}|$$

여기서:
- $e_i = y_i - y_{R,i}$: 추적 오차
- $\max e_i - \min e_i$: 웨이퍼 내 부균일성(WIWNU) 벌칙
- $a_i - a_{i-1}$: 압력 변화 평활성 항

파라미터: $\alpha_i = $, $\beta = 0.01$[2][1]

#### 2.2.3 DSAC-T 알고리즘

DSAC-T는 DSAC의 세 가지 개선사항을 적용합니다.[1]

**1) 비판가 기울기 조정 (Critic Gradient Adjustment)**

분포형 목표값 대신 비분포형 목표값으로 대체하여 안정성 향상:

$$T_QQ(s',a') = r + \gamma Q_{\bar{\psi}}(s',a') - \alpha \log \pi_{\phi}(a'|s')$$

조정된 기울기:

$$\nabla_{\psi} \mathcal{J}_Z^* = \mathbb{E} \left[ \frac{\nabla_{\psi}(T_Z(s,a) - Q_{\psi}(s,a))^2}{2\sigma^2_{\psi}(s,a)} \cdots \right]$$

**2) 쌍둥이 값 분포 학습 (Twin Value Distribution Learning)**

두 개의 값 분포를 학습한 후 더 작은 값 선택:

$$\mathcal{J}_Z^{min} = \mathbb{E}_{s,a,r,s'} \left[ \frac{(T_Z(Q^{\min}(s,a)) - Q_{\psi}(s,a))^2}{2\sigma^2_{\psi}(s,a)} \cdots \right]$$

$$J_{\pi} = \mathbb{E}_{s \sim D} \left[ \min_{i=1,2} Q_i(s,a) - \alpha \log \pi_{\phi}(a|s) \right]$$

이는 과대 추정 편향을 경감하면서 약간의 과소 추정을 감수합니다.[1]

**3) 분산 기반 클리핑 (Variance-Based Target Return Clipping)**

고정된 경계 대신 분산 기반 동적 경계 사용:
$$l = -\xi\sigma(s,a), \quad h = \xi\sigma(s,a)$$

여기서 $\sigma(s,a)$는 값 분포의 표준편차, $\xi$는 클리핑 척도 제어 상수입니다.[1]

### 2.3 성능 향상 메커니즘

#### 2.3.1 Q-값 과대 추정 완화

분포형 학습은 기존 Q-러닝의 과대 추정 문제를 해결합니다:

$$\mathbb{E}[Z(s,a)] = Q(s,a)$$

분포의 분산이 크면 Q-값 업데이트의 단계 크기가 감소하여 안정성이 향상됩니다.[1]

#### 2.3.2 하이퍼파라미터 튜닝 민감도 감소

분산 기반 클리핑으로 보상 스케일에 대한 의존성을 감소시켜, 수동 조정의 필요성을 크게 줄였습니다.[1]

#### 2.3.3 비선형 공정 적응

1DCNN-LSTM VM 모델은 시계열 특성을 효과적으로 포착하여 공정 드리프트와 변이에 신속하게 대응합니다.[1]

### 2.4 실험 결과: 성능 비교

| 제어 방법 | 고정 참조값 환경 | 동적 참조값 환경 | 변이 후 응답 |
|---------|-------------|-------------|----------|
| **DSAC-T** | **최우수** | **최우수** | 10회 이내 안정화 |
| DSAC | 우수 | 중간 | 불안정 |
| SAC | 중간 | 중간 | 느린 수렴 |
| TD3 | 중간 | 약함 | 진동 |
| DDPG | 약함 | 약함 | 심한 진동 |

**RMSE 성능**: DSAC-T는 SAC 대비 **15-25% 개선**, 전통적 제어 대비 **30-40% 개선**을 달성했습니다.[1]

## 3. 모델의 일반화 성능 향상 가능성

### 3.1 일반화 성능의 현재 상태

논문 결과에서 **DSAC-T의 일반화 능력**은 다음과 같이 평가됩니다:[1]

#### 성공 사례:
- **고정 참조값 ( $y_{r,i} = 4$ )**: RMSE 0.62 달성, 모든 알고리즘 중 최우수
- **동적 참조값 ( $y_{r,i} \sim N(0, \sigma)$ )**: RMSE 1.12-1.84, DSAC 대비 25% 향상
- **변이 대응**: 1회 변이 후 10회 폴리싱 내 안정화

#### 한계점:
- **Zone 2 문제**: 가장자리 영역 근처의 Zone 2는 압력 제약으로 인해 제어 성능이 저하됨 (RMSE 1.84)
- **참조값 범위 의존성**: 참조값 범위가 클수록 성능 저하 (범위 6σ에서 약 20% 성능 감소)
- **데이터 희소 영역**: MRR이 범위 끝단인 영역에서는 훈련 데이터가 적어 일반화 성능 저하[1]

### 3.2 일반화 성능 향상을 위한 미래 방향

#### 3.2.1 데이터 확충의 중요성

논문에서 명시적으로 제시한 개선 방향:
$$\text{성능 저하} = f(\text{데이터 희소성})$$

더 광범위한 데이터 수집이 **가장 중요한 개선 요인**입니다. 특히:
- 극단값(edge case) 영역 데이터 확충
- 다양한 공정 상태 데이터 수집
- 계절, 시간대별 환경 변화 데이터 포함[1]

#### 3.2.2 멀티존 상호작용 모델링

**Zone 5의 다른 존에 대한 영향을 고려하지 않은 문제**:

현재 모델:
$$y_{i,t} = f(y_{5,t}, p_{i,t}, U_t), \quad i \neq 5$$

개선 방향:
$$y_{i,t} = f(y_{5,t}, \{p_{j,t}|j \text{ adjacent}\}, U_t)$$

인접 존들의 압력 상호작용을 포함하면 Zone 2 성능을 15-20% 개선할 수 있을 것으로 예상됩니다.[1]

#### 3.2.3 고차원 상태 표현

현재 상태 공간의 확장:
- 드릴 온도, 습도 등 환경 변수 추가
- 패드 표면 거칠기, 연마액 조성 등 물리적 변수 포함
- 히스토리 길이 조정 (현재 k=14 time steps)

이러한 확장은 **전이 학습(transfer learning) 능력을 향상**시킬 것입니다.[1]

#### 3.2.4 도메인 적응 기법

서로 다른 CMP 장비나 공정 조건으로의 전이:
$$Q_{\text{target}} = Q_{\text{source}} + \Delta Q_{\text{adapt}}$$

**메타 학습(meta-learning)** 기법을 적용하여 새로운 공정 환경에 신속하게 적응할 수 있는 기제를 개발할 수 있습니다.[1]

#### 3.2.5 분산형 가치 함수의 활용

현재의 가우시안 분포 가정을 완화:
$$Z(s,a) \sim \mathcal{N}(Q(s,a), \sigma^2(s,a))$$

**다봉 분포(multimodal distribution)**를 지원하는 구조로 확장하면, 복잡한 이중 안정성(bistability) 현상을 더 잘 모델링할 수 있을 것입니다.[1]

## 4. 논문의 연구 영향 및 향후 연구 고려사항

### 4.1 학술 및 산업적 영향

#### 4.1.1 이론적 기여
- **모델-프리 제어의 확증**: 반도체와 같은 극도로 복잡한 산업 공정에서도 모델-프리 DRL이 실행 가능함을 최초 입증
- **분포형 RL의 실제 응용**: DSAC-T가 실제 제조 환경(시뮬레이션이 아닌 실제 데이터 기반)에서 우수하게 작동함을 보여줌
- **멀티존 제어의 새로운 패러다임**: 기존의 영역별 독립 제어에서 **상호작용 기반 제어**로의 전환 가능성 제시[1]

#### 4.1.2 산업 실무 영향
- **제어 시스템 현대화**: 기존의 EWMA, d-EWMA 등 선형 제어 알고리즘의 대체 가능성 제시
- **적응형 제어**: 공정 업그레이드 시 모델을 재구축하지 않고 재훈련만으로 대응 가능
- **생산 비용 감소**: CMP 공정 품질 향상으로 인한 수율 증가 및 스크랩 감소
- **이설 시간 단축**: 새로운 공정 도입 시 하이퍼파라미터 튜닝 시간 대폭 감소[1]

### 4.2 향후 연구 시 고려할 점

#### 4.2.1 온-정책 알고리즘의 탐색

논문의 제한사항:
> "이 논문은 오프-정책 기울기 방법만 논의합니다. TRPO, PPO 같은 온-정책 알고리즘에 대한 추가 연구가 필요합니다."[1]

**고려사항**:
- TRPO의 신뢰 영역(trust region) 기법이 강화된 안정성 제공 가능
- PPO의 낮은 계산 비용이 실시간 제어에 유리
- 하이브리드 접근법: 온-정책과 오프-정책의 장점 결합[1]

#### 4.2.2 멀티존 압력의 상호작용 모델링

현재 가정:
> "Zone 5의 MRR에 대한 다른 존의 압력 영향을 이 논문에서 무시했으며, 향후 연구에서 고려할 수 있습니다."[1]

**구체적 접근**:
- 물리 기반 모델과 데이터 기반 모델의 하이브리드
- 그래프 신경망(GNN)을 이용한 공간적 상호작용 학습
- 음향 방출(AE) 센서 데이터 활용으로 비선형 동역학 포착[1]

#### 4.2.3 다른 반도체 공정으로의 확장

논문 제안:
> "제어 전략의 배포는 CMP 외에도 식각(Etch), 리소그래피(Litho) 등 다른 반도체 제조 공정에도 적용될 수 있으며, 이는 가치 있는 연구 방향이 될 것입니다."[1]

**고려 사항**:
- 각 공정의 물리적 특성과 제어 특이성 분석
- 전이 학습(transfer learning)을 통한 신속한 적응
- 멀티 에이전트 강화학습으로 fab 레벨 최적화[1]

#### 4.2.4 실시간 배포 전략

현재 시뮬레이션 기반 검증에서 실제 fab 적용으로의 전환:

**단계적 접근**:
1. **시범 영역(Pilot Area)**: 소규모 공정에서 6-12개월 검증
2. **점진적 확대**: 성공 사례 기반으로 라인 확장
3. **모니터링 시스템**: 온라인 성능 추적 및 실시간 재훈련
4. **폴백 메커니즘**: 제어 불안정 시 즉각 기존 방식으로 복귀[1]

#### 4.2.5 데이터 품질 및 개인정보 보호

**기술적 고려**:
- 노이즈가 큰 센서 데이터의 전처리 개선
- 결실된 데이터 처리 (일부 wafer는 측정 건너뜀)
- 합성 데이터(synthetic data) 생성으로 희소 영역 보강

**정책적 고려**:
- 공정 데이터의 기술 보안 확보
- 다중 fab의 데이터 통합 시 개인정보 보호 방안[1]

#### 4.2.6 경제성 분석

실제 도입을 위한 투자 수익률(ROI) 분석:

**정량적 효과**:
- 수율 향상: 2-5% 개선 가능
- 스크랩 감소: fab당 연 수억 원 절감
- 공정 안정성: 편차 감소로 신뢰성 향상

**정성적 효과**:
- 엔지니어 업스킬(upskill) 기회
- 데이터 기반 의사결정 문화 정착
- 경쟁력 강화[1]

### 5.2 논문의 상대적 우수성

#### 5.2.1 기술적 우위

**DSAC-T 알고리즘의 혁신**:

```
기존 방법들:
DDPG (2015) → TD3 (2018) → SAC (2019) → DSAC (2022) → [논문의] DSAC-T (2025)
  ↓           ↓           ↓         ↓              ↓
 기본 DRL  지연 업데이트  엔트로피  분포형 값   세 가지 개선
```

**구체적 개선사항**:
- **Q-값 과대 추정 감소**: 기존 대비 $$\text{variance}(\Delta Q) \downarrow 30-40\%$$
- **하이퍼파라미터 민감도**: 보상 스케일 $\times 2$-3배 변화에도 안정적 성능 유지
- **수렴 속도**: SAC 대비 20-25% 빠른 수렴[1]

**VM 모델의 혁신**:

기존: 선형 MISO 모델, RMSE ~5-8%
본 논문: 비선형 1DCNN-LSTM, RMSE ~2-3%, **60% 이상 오차 감소**[1]

#### 5.2.2 실무 적용성

| 차원 | 기존 방법 | 본 논문 | 개선도 |
|------|----------|--------|--------|
| **적응 시간** | 일주일 이상 | 1시간 내 | **168배 빠름** |
| **멀티존 제어** | 단순 비례 제어 | 상호작용 고려 | **완전히 새로운 접근** |
| **환경 변이 대응** | 수동 재조정 | 자동 재학습 | **완전 자동화** |
| **엔지니어 개입** | 높음 | 거의 불필요 | **80% 감소** |

#### 5.2.3 평가 방법론의 우수성

기존 연구: 단순 선형 모델 시뮬레이션
본 논문: **실제 fab 데이터 기반 비선형 VM 모델 + 4가지 변이 시나리오**[1]

**변이 유형**:
1. 백색 잡음 (시뮤레이션 복잡도 낮음)
2. 공정 드리프트 (현실성 높음)
3. 무작위 변이 (예측 어려움)
4. 동적 목표값 (산업 실무 정확함)[1]

## 6. 결론 및 종합 평가

### 6.1 학술적 기여도

본 논문은 **세 가지 측면에서 획기적인 공헌**을 합니다:

1. **이론적 기여**: 분배형 강화학습이 극도로 복잡한 산업 공정에 효과적으로 적용될 수 있음을 최초 입증
2. **기술적 기여**: DSAC-T의 세 가지 개선사항이 실제 공정 환경에서 하이퍼파라미터 튜닝 민감도를 획기적으로 감소시킴
3. **방법론적 기여**: 비선형 VM 모델과 DRL의 결합이 새로운 연구 패러다임을 제시[1]

### 6.2 산업 적용 가능성

**즉시 적용 가능 (1-2년)**:
- CMP 공정 제어의 실제 fab 배포
- 수율 향상 및 비용 절감
- 자동화 수준 향상

**조건부 적용 (2-3년)**:
- 다른 반도체 공정으로의 확대
- 메타-학습을 통한 신속 적응화
- 실시간 모니터링 시스템 통합[1]

### 6.3 향후 연구자들을 위한 조언

1. **데이터 수집**: 극단값 영역의 데이터 확충이 일반화 성능의 핵심
2. **모델 개선**: 멀티존 상호작용을 명시적으로 모델링
3. **실무 연계**: 학계의 기법을 fab 현장 조건에 적응시키는 과정이 가장 도전적
4. **인터디시플린 연구**: 공정 물리학자, 제어 공학자, AI 연구자의 협력 필수[1]
